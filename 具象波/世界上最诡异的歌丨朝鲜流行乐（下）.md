
# 世界上最诡异的歌丨朝鲜流行乐（下）

世界上最诡异的电影不在德国，而是在朝鲜。这种诡异的源头就是四个大字，主体思想。

十几年前，网络上出现一种被叫做蒸汽波的风格。简单来说就是把日本、美国80年代的流行歌改成更慢速、更扎音质的版本，勾起大家的童年回忆。借着这波潮流，网上还二创了另一个类似的版本，同样是慢速加扎音质。但原曲不是日本流行，而是朝鲜流行，大家管这个叫主体播。

没错，主体思想就是朝鲜唯一且正确的slogan。可啥是主体思想呢？其实朝鲜音乐的发展就是一部主体思想发展史。

上世纪五六十年代，也就是金家一代金日成时期，提出了主体思想的概念，朝鲜出现了各种革命歌剧。西方美声唱法加东方五声音阶，歌曲听起来是西方管弦大乐队，但故事情节全部服务于朝鲜领导人。军民从不走样。此后，朝鲜乐坛的半壁江山都在歌颂金家。

后来朝鲜的音乐一直在进化。从喜爱电影和音乐的金正日，到留学瑞士归来的金正恩，从电子到摇滚，每一任朝鲜领导人都想用音乐来证明：你们真的误会了，朝鲜才不是文化沙漠，我们很先进。

当朝鲜音乐在YouTube上获得了百万播放量，朝鲜领导人更加觉得"黑子说话"，我们的先进可是得到了世界人民的认可。殊不知，大家只是把朝鲜音乐当乐子。

为什么朝鲜音乐会变成笑话呢？其实只听一两秒钟的话，朝鲜歌真的黑不了，还是挺潮的。可是这一两秒过后，就是随之而来的朝鲜土味——神仙开头加土味冲击。这就是朝鲜特色流行乐。

这是因为朝鲜音乐与时俱进地引进了合成器，学习摇滚和电子音乐，但他们却无法抛弃苏联式的管弦乐思维，以及行军乐队的同款2拍子。这种二拍子就是土味的来源，听起来就是"嘣加嘣加"，非常洗脑，就像行军的脚步、挥舞的锄头，非常有能量。唯一的缺点就是很容易让人误以为是村头的红白丧葬乐队。

朝鲜并不喜欢黑人那种切分音或者摇摆的感觉，而是偏爱一种绝对精确、机械稳定的感觉。它反映了朝鲜的主体思想，即每个人都应该是螺丝钉，共享金家的大脑。这些没有技巧、全靠感情诠释的唱法，让朝鲜音乐变得土土的。

但在朝鲜，音乐一直很重要。如果你有音乐天赋，就会被送到平壤最好的音乐学校，优先住房、接触权利，逆天改命，所以竞争非常的激烈。金正恩的妻子李雪主就曾经是一个管弦乐队的成员。

朝鲜音乐之所以诡异，就是因为它其实是一个"画皮"的故事。它把摇滚和电子的个性、反叛、解构的内脏统统掏出来扔掉，只剩下一张曲风的皮。这些皮都必须且仅能套在主体思想上。也就是说，打从一开始，朝鲜音乐就压根没想过要服务于个人表达。

这一点更体现在朝鲜的国民乐器——手风琴上。在朝鲜，并不是说你喜欢手风琴，而是几乎所有人都要学手风琴。手风琴不需要外部供电，又旋律又和弦，一把手风琴就是一个乐队，自给自足，服务集体。毫不夸张地说，手风琴就是主体思想的完美载体。

博主听朝鲜音乐的时候，就经常忍不住想：那些热情洋溢、训练成机器般表演的乐手，那些在台下整齐划一鼓掌的观众，他们真的喜欢这种音乐吗？人真的可以舍弃掉作为个体的一切情感，仅仅靠宏大的叙事而活着吗？

或许我永远都不会知道真正的答案，但朝鲜音乐无疑是这世界上最有想象力的声音。今天全世界的流行音乐都不可避免地被西化，然而朝鲜音乐却一直拼命地驯服西方音乐。他们把外来的乐器和曲风都驯化成手风琴一样的主体思想工具。

在这个人人都标榜自由和个性的年代里，朝鲜却通过抹杀个性换取了一种另类的自由。在这个连叛逆都显得千篇一律的时代，朝鲜以一种常人难以想象的方式，完成了一场最诡异的叛逆。
